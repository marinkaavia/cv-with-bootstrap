U-Check with Face Recognition (UFace)

UFace is an innovative attendance system using face recognition technology to seamlessly connect teachers, administrators, and students without the need for personal devices.

Attendance checking system with face recognition (UFace)

General Description:

U-Check with Face Recognition, also known as UFace, is a technological solution designed to streamline the attendance and identity verification processes in educational settings. This project eliminates the need for individual phones by incorporating face recognition technology, enabling a more efficient connection among teachers, administrators, and students. The system's simple interface ensures ease of use and quick adoption, enhancing daily administrative operations and improving security measures.

Technologies Used:

Python: were used numpy and OpenCV libraries

OpenCV (cv2 library): Utilized for capturing and processing images from video streams, this library is instrumental in implementing face detection algorithms.

september-december 2023

Furniture & interior design content creation (Home designAR)

Home designAR - create 2D and 3D furniture and decoration content by inputting text or images and overlay them in real space using AR technology

Furniture & interior design content creation (Home designAR)

General Description:

This innovative project allows users to create and customize 2D and 3D furniture and decoration items simply by inputting text or images. Leveraging augmented reality (AR) technology, users can overlay these items in real space, transforming how interior design is visualized and planned. The tool offers functionality to customize colors, add furniture to a room, and apply realistic rendering effects, enhancing the user experience and design accuracy.

Technologies Used:

Figma: Used for designing the UI/UX elements and prototyping the application interface.

HTML & CSS: Employed to structure and style the web application, ensuring it is visually appealing and functional.

JavaScript (JS): Drives the interactive elements of the application, handling AR integrations and dynamic content customization.

Period:

march-june 2024

Educational AI-Powered Mathematics Learning web-site

Application aims to make mathematics engaging for students preparing for the Russian state exam, create a responsive and effective educational experience.

Educational AI-Powered Mathematics Learning web-site for Russian State Exam

General Description:

This interactive web application is designed to help students master advanced mathematics in preparation for the Russian state exam. By incorporating artificial intelligence (AI), the tool provides personalized learning experiences, offering problems and explanations tailored to each student's needs. The application focuses on improving problem-solving skills and understanding key mathematical concepts essential for the exam.

Technologies Used:

Python: The core programming language used for backend development, enabling the integration of AI functionalities and data handling.

AI Libraries (such as NumPy, Pandas, SciPy): These libraries are crucial for processing mathematical data, generating test problems, and analyzing student performance to tailor the learning experience.
